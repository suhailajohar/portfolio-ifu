Technical Writing Portfolio

This portfolio presents sample user-facing documentation written in a topic-based structure, reflecting practices used in regulated medical device environments.

Portfolio Contents

Document Description
Instructions for Use (IFU) Core device IFU demonstrating intended use, warnings and precautions, operating instructions, troubleshooting, and symbols.
Understanding Your Activity Dashboard Companion user education topic explaining weekly activity monitoring and trend interpretation with reference to publicly available health guidance.

Scope and Intent

These documents are fictional samples created for portfolio demonstration purposes. They are intended to illustrate controlled language, structured authoring, and user-centered instructional design.

No diagnostic, therapeutic, or medical claims are made.
