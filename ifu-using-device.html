Instructions for Use (IFU)

Wellness Activity Monitor

1. Purpose

The Wellness Activity Monitor is intended to record daily physical activity, sleep duration, and estimated calorie expenditure to support user awareness of personal wellness trends.

2. Intended Use

The Wellness Activity Monitor is intended for use by adults to track and review personal wellness data through a companion mobile application.

3. Contraindications

No known contraindications.

4. Warnings and Precautions

WARNING
Failure to follow the instructions in this document may result in inaccurate interpretation of data.
CAUTION
The Wellness Activity Monitor does not provide medical diagnoses and should not be used as a substitute for professional medical advice.

5. Device Description

The Wellness Activity Monitor consists of:

  • A wearable activity tracking device
  • A rechargeable battery
  • A companion mobile application

The device records step count, sleep duration, and estimated calorie expenditure based on detected user activity.

6. Operating Instructions

6.1 Before First Use

  1. Fully charge the device using the supplied charging cable.
  2. Download the companion mobile application.
  3. Create a user profile following the in-application instructions.

6.2 Daily Use

  1. Wear the device as instructed during daily activities.
  2. Synchronize the device with the mobile application at least once per day.
  3. Review activity summaries within the application dashboard.

7. Interpreting the Data

Activity Data

Daily step counts represent the total number of steps detected during wear time.

Sleep Data

Sleep duration represents an estimate of time spent asleep based on detected movement patterns.

Calorie Data

Calorie expenditure is an estimate derived from activity data and user profile information.

8. Troubleshooting

Issue Possible Cause Recommended Action
Device does not synchronize Bluetooth disabled Enable Bluetooth and retry synchronization
No activity data recorded Device not worn Ensure the device is worn correctly
Inaccurate readings Improper device positioning Adjust device fit and position

9. Storage and Handling

  • Store the device in a clean, dry environment.
  • Avoid exposure to extreme temperatures.
  • Keep out of reach of children.

10. Disposal

Dispose of the device in accordance with local regulations for electronic waste.
